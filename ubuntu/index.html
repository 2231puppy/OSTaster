Ubuntu is a very popular linux distro used for enterprise applications, development, and desktop use. It is well known, and the latest LTS version is 20.04 and runs decently on my tester laptop. However, I noticed an occasional visual glitch and Firefox loading times above 1 second, so therefore, I cannot reccommend it on very low-spec machines. However, if your machine can run Windows 10 smoothly and you are looking for a good Linux alternative, this is the OS for you.
